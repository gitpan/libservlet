NAME

Servlet::Http::HttpSessionBindingListener - session binding listener interface

SYNOPSIS

  $listener->valueBound($event);
  $listener->valueUnbound($event);

DESCRIPTION

This listener interface causes an object to be notified when it is bound to or unbound from a session. The object is notified by an object implementing Servlet::Http::HttpSessionBindingEvent. This may be as a result of a servlet programmer explicitly unbinding an attribute from a session or due to a session being invalidated or timing out.

METHODS

valueBound($event)
Notifies the object that it is being bound to a session.

Parameters:

$event
an instance of Servlet::Http::HttpSessionBindingEvent

valueUnbound($event)
Notifies the object that it is being unbound from a session.

Parameters:

$event
an instance of Servlet::Http::HttpSessionBindingEvent
