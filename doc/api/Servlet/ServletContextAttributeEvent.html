NAME

Servlet::ServletContextAttributeEvent - servlet context attribute event class

SYNOPSIS

  my $event = Servlet::ServletContextAttributeEvent->new($context, $name,
                                                         $value);
  my $name = $event->getName();
  my $object = $event->getValue();
  my $context = $event->getServletContext();
  # or
  my $context = $event->getSource();

DESCRIPTION

This is an event class for notifications about changes to the attributes of the servlet context of a web application.

CONSTRUCTOR

new($context, $name, $value)
Construct an instance for the given context, attribute name and attribute value.

Parameters:

$context
the servlet context that is the source of the event

$name
the name of the affected servlet context attribute

$value
the value of the affected servlet context attribute

METHODS

getName()
Returns the name of the affected servlet context attribute.

getServletContext()
Returns the Servlet::ServletContext object that is the source of the event.

getSource()
Returns the Servlet::ServletContext object that is the source of the event.

getValue()
Returns the value of the affected servlet context attribute. If the attribute was added, this is the value of the attribute. If the attribute was removed, this is the value of the removed attribute. If the attribute was replaced, this is the old value of the attribute.
