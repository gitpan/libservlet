NAME

Servlet::ServletContextEvent - servlet context event class

SYNOPSIS

  my $event = Servlet::ServletContextEvent->new($context);
  my $context = $event->getServletContext();
  # or
  my $context = $event->getSource();

DESCRIPTION

This is an event class for notifications about changes to the servlet context of a web application.

CONSTRUCTOR

new($context)
Construct an instance for the given context.

Parameters:

$context
the servlet context that is the source of the event

METHODS

getServletContext()
Returns the Servlet::ServletContext object that is the source of the event.

getSource()
Returns the Servlet::ServletContext object that is the source of the event.

SEE ALSO

the Servlet::ServletContext manpage, the Servlet::ServletContextAttributeListener manpage
